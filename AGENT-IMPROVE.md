# Agent Self-Improvement Protocol

After completing ANY task, you MUST do the following:

## Step 1 — Review What You Did
- What commands did you use?
- What patterns did you follow?
- What problems did you solve?
- What decisions did you make?

## Step 2 — Check Against AGENTS.md
- Is this knowledge already in AGENTS.md?
- If yes → skip, do nothing

## Step 3 — Apply the Filter (STRICT)
Only add/modify AGENTS.md if ALL of these are true:

✅ Future-proof → Will this rule apply 6+ months from now?
✅ Necessary → Will AI make mistakes without this rule?
✅ Secure → Does this improve safety / prevent bad decisions?
✅ Reusable → Will this apply to more than one task?

If ANY answer is NO → do not touch AGENTS.md

## Step 4 — If Adding
- Add under the correct section
- Write it as a clear rule, not a story
- Maximum 2 lines per rule
- No duplicates

## Step 5 — Report
Tell the user:
- "Added to AGENTS.md: [what and why]"
- OR "No changes needed to AGENTS.md"
